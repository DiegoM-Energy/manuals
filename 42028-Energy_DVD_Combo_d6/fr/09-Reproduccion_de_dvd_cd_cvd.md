## 7. LECTURE DVD/CD/VCD

> Avis: Cet appareil n'est pas compatible avec le format DIVX3.

* Ouvrir/Fermer

	Appuyer ce bouton pour ouvrir le plateau du disque. Appuyez à nouveau ce bouton pour femer le plateau du disque. La lecture du disque commencera automatiquement.

	> Avis: Introduire le disque avec la face imprimée vers le haut.

* Play/Pause

	Appuyez ici pour commencer la lecture. Appuyez à nouveau ce bouton pour arrêter la lecture pour l'instant.

* Slow (Caméra ralenti)

	Appuyez sur SLOW pour activer le ralenti en avant. Quand vous appuyez les boutons, la vitesse diminuiera dans l'ordre suivant: 1/2, 1/4, 1/8, 1/16, 1/32. Pour rétablir la vitesse normal de lecture, appuyez sur Play/Pause.

	> Avis: La fonction de lecture ralenti ne sera pas disponible pendant la lecture d'un CD.

* Avance / recul rapide

	Appuyez sur FF/FB pour faire une recherche rapide en arrière ou en avant respectivement. Pour rétablir la vitesse normlae de lecture, appuyez sur Play/Pause. Chaque fois que vous appuyez sur ce bouton, la vitesse de lecture sera modifiée dans cet ordre: x2, x4, x8, x16, x32.

* Antérieur/Suivant

	Pendant la lecture, appuyez sur reculer pour aller vers le début du chapître ou chanson antérieure/suivante. Pendant la lecture d'un CD audio, utilisez les boutons numériques pour introduire directement le numéro de la chanson que vous voulez écouter.

* Arrêter

  * Appuyez une seule fois le bouton Stop pour arrêter la lecture.
  * Appuyez à continuation sur Play/Pause pour recommencer la lecture à partir du point d'interruption.
  *Appuyez deux fois sur le bouton Stop pour arrêter définitivement la lecture.
    Vous ne pourrez pas recommencer la lecture à partir du point d'arrêt.
  * Appuyez sur Play/Pause pour commencer la lecture au début du disque.

* Menu

	Appuyez pour visualiser le menu du DVD. Ce bouton n'est pas operationnel pendant la lecture d'un CD audio.

* Titre

	Appuyez pour visualiser le menu des titres d'un DVD (la disponibilité de cette fonction dépend du disque).

* Zoom

	Augmentez la taille de l'image pendant la lecture. Pendant la lecture d'un disque, la taille de l'image est modifiée quand vous appuyez sur ces boutons: x1 -> x2 -> x4.

	Utilisez les boutons directionnels pour se déplacer dans l'image agrandie. Pour rétablir la taille normale de l'image, appuyez sur Zoom jusqu'à visualiser le message Zoom x1 sur l'écran.

* Angle

	Visualisez une scène depuis les différents angles (la disponibilité de cette fonction dépend du DVD).

* Audio

	Choisir une langue pour l'audio, disponible dans le disque. Avec une seule pulsation, visualisez la langue actuelle et appuyez sur les langues disponibles. Avis: Les options disponibles après avoir réproduit VCD sont: L/L (Left), R/R (Right), L/R (Mix).Si le format audio est DTS n'aura pas aucune sortie audio.

* Surtitres

	Choisir l'un des surtitres disponibles dans le disque. Pendant la lecture, changez la langue des surtitres ou désactivés. Avec une seule pulsation, visualisez la langue actuelle et appuyez pour changer entre les différents surtitres disponibles.

* Info

	Pendant la lecture d'un disque, appuyez sur Info pour voir l'information sur la lecture en cours.

* GoTo

	Choisir un titre/chapître d'un DVD ou un moment concret dans un CD/DVD. Appuyez sur la Gauche/Droite pour choisir le titre / chapître (DVD) ou temps (CD/VCD) et introduire la valeur avec les boutons numériques. Appuyez OK pour confirmer. La lecture du fragment choisi commencera.

	> Avis: Certains disques ne permettent pas d'utiliser cette fonction. Pour VCDs, cette fonction est seulement disponible si PBC est débranché.

* Lecture Programmée

	Le d5 AV permet de choisir l'ordre de lecture des titres chapîtres ou chansons. Avis: Pendant la lecture d'un VCD ou d'un SVCD, la lecture programmée sera seulement disponible si la fonction PBC est désactivée. Utilisez les boutons Haut/Bas pour choisir un titre à la droite du menu de programmation et appuyer OK pour confirmer. Utilisez les boutons Haut/Bas pour choisir un chapître et appuyer sur OK pour confirmer. Le numéro du programme choisi apparaitra dans la partie gauche du menu. Répétez les démarches antérieures pour le reste des titres/chapîtres (ou chansons d'un CD). Appuyez sur Play/Pause ou OK pour commencer la lecture programmée. Pour effacer un fichier choisi et appuyer sur Stop.

* Répétition

	Le d5 AV permet de répéter la lecture d'une chanson, un chapître ou une piste.

	> Avis: Pendant la lecture d'un VCD/SVCD la répétition sera seulement disponible si la fonction PCB est débranchée.

* Bouton REPEAT:

    * DVDs: Débranché > Chapître > Chanson
    * CD /VCD/SVCD: Désactivé > Piste > Tout
    * Fichiers MP3 / JPEG / WMA: Désactivé > Fichier > Dossier > Tout

* Bouton A-B:

    * Appuyez sur le bouton pour établir le début du segment (point A).
    * Appuyez sur A-B à nouveau pour établir la fin du segment (point B).

	Le segment choisi sera répété jusqu'à appuyer à nouveau sur A-B.

	Avis: Cette fonction pourra être utilisée dans le même titre ou piste. Si les points A et B sont dans des titres ou pistes différentes, la fonction sera annulée.
